# 🌦️ Weather App

A modern Weather Application built using **React + Vite** that allows users to search for real-time weather information for any city.

🔗 Live Demo  
https://weather-app-t7oh-le8y8mdhk-vinays-projects-cfccd845.vercel.app/

---

## 🚀 Features

- 🌍 Search weather by city name
- 🌡 Shows temperature in Celsius
- 💧 Displays humidity
- 💨 Displays wind speed
- ☁️ Weather condition description
- ⚡ Fast performance using Vite
- 📱 Responsive design

---

## 🛠️ Technologies Used

- React
- Vite
- JavaScript
- CSS
- Weather API
- Vercel (Deployment)
